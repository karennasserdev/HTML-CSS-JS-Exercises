

Um site optimizado para dispositivos móveis, deve conter o meta
com acesso a viewport:

            <meta name="viewport" content="width=device-width, initialscale=1.0, user-scalable=no">

A propriedade width controla o tamanho da viewport;
A propriedade initial-scale controla o nível de zoom quando a página é
carregada;
User-scalable controlam a permissão para o usuário efetuar aumento
ou diminuição da página;

------

Muitos elementos <meta> incluem atributos de name e content:

<meta name="author" content="Gonçalo Vieira">
<meta name="description" content="Html5, js, css front…”>
<meta name="keywords" content="sites, responsive, html, html5">
<meta name="rating" content="General">
<meta name="revisit-after" content="5 days”>

O Meta Robots - meta name="robots"
Esta Meta Tag foi criada para passar orientações aos robots dos motores
de pesquisa.

• index: indexe esta página - aparecer nos resultados da pesquisa;
• noindex: não indexe esta página - não aparecer nos resultados de busca. Útil para páginas como de login e acesso à intranet;
• follow: seguir os links da página para descobrir novas páginas.
 nofollow: nenhum dos links desta página deve ser seguido;
• noarchive: Não exibir a versão em cache da página;
• noimageindex: não indexe nenhuma imagem da página.

exemplo:
                <meta name="robots" content="index, follow">
                
- sintaxe mais comum de meta robots - orienta os motores de pesquisa a
indexar o conteúdo da página e seguir todos os links para descobrir
novas páginas
